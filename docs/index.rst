InstanSeg documentation
=====================

InstanSeg is a Python package for semantic segmentation of brightfield and fluorescence microscopy images. In particular, it allows for simultaneous cell and nucleus segmentation of fluorescence images with an arbitrary number of channels.

Take a look at the `GitHub page <https://github.com/instanseg/instanseg>`_ of the project for more details (such as more information on the project and how to install it).
